# 주의깊게 살펴본 부분
## 회원가입 및 메일인증

- 이메일과 비밀번호를 이용하여 간단하게 회원가입을 하는 API를 만들었습니다.
- SMTP를 이용하여 구글 계정 또는 마이크로소프트 메일 서버로 부터 인증 메일을 보낼 수 있도록 메일인증 API를 만들었습니다.
- 예상 고객 분들이 개인 정보에 매우 민감한 분들일 가능성이 높기 때문에 메일주소나 회원 id 같은 정보는 AES256를 이용하여 암호화하였습니다.
